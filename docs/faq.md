# FAQ

## Can I store real Clash subscription links here?

No. This is a public repository. Use placeholders only.

## Where should private files go?

Use local ignored paths such as `private/`, `secrets/`, or ignored files under `subscriptions/`.

## Why is there a security check workflow?

It helps catch obvious accidental leaks, but it does not replace manual review.

## What is the backup branch for?

The `backup` branch is intended as a known-good recovery point before larger changes.
