## Directory Purpose

- `docs/`: documentation and usage notes
- `rules/`: public/safe rule examples
- `templates/`: sanitized Clash templates
- `subscriptions/`: local private subscription files are ignored by default
- `scripts/`: maintenance scripts
- `backup/`: backup policy notes
