# Public Repository Safety

Because this repository is public, treat it as visible to everyone.

## Never Commit

- Real Clash subscription URL
- Real proxy node address if it is private
- Passwords
- Tokens
- API keys
- Personal account details

## Safe to Commit

- Documentation
- Placeholder examples
- Sanitized templates
- Public rule notes

## If Sensitive Data Is Accidentally Pushed

1. Remove the sensitive data immediately.
2. Rotate or regenerate the exposed secret/subscription.
3. Review commit history if necessary.
