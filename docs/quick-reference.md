# Quick Reference

## Clone

```bash
git clone https://github.com/liyan-lucky/clash-subscription.git
cd clash-subscription
```

## Review Changes

```bash
git status
git diff
```

## Commit

```bash
git add .
git commit -m "Update Clash templates"
git push origin main
```

## Update Backup Branch

```bash
bash scripts/backup-branch.sh
```
