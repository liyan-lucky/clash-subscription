# Clash Notes

## Common Ports

- Mixed port: `7890`
- HTTP port: `7890` or `7891`
- SOCKS port: `7891`
- External controller: `9090`

Adjust ports according to your local environment.

## Common Modes

- `rule`: route traffic by rules
- `global`: route traffic through selected proxy
- `direct`: bypass proxy

## Public Repository Rule

Only publish sanitized templates and rule examples. Keep real node information private.
